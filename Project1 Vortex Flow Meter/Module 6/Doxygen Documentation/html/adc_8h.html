◆ ADC_Custom_Config()

void ADC_Custom_Config ( void  )

Custom configuration for the ADC module.

The function sets up the ADC for continuous conversion mode and configures it to measure from the on-chip temperature sensor. It also sets the sampling time and starts the ADC conversion.

◆ readADC()

void readADC ( void  )

Read ADC samples and compute the frequency based on zero crossing.

The function counts the number of zero crossings in the provided samples and calculates the frequency based on the sample time and the number of samples.
