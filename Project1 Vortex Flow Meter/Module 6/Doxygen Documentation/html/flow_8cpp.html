void calculate_flow ( void  )

Calculates the fluid flow based on temperature, density, and viscosity.

Calculates the fluid flow.

The function reads the temperature and based on it calculates the fluid's density and viscosity. Using these values and the Reynold's number, it computes the Strouhal number and uses it to adjust the velocity. Finally, it calculates and sets the flow rate.
