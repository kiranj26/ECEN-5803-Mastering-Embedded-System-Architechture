Checks whether the specified RNG interrupt has occurred or not.

Parameters
__HANDLE__RNG Handle
__INTERRUPT__specifies the RNG interrupt source to check. This parameter can be one of the following values:
  • RNG_FLAG_DRDY: Data ready interrupt
  • RNG_FLAG_CECS: Clock error interrupt
  • RNG_FLAG_SECS: Seed error interrupt
Return values
Thenew state of RNG_FLAG (SET or RESET).
