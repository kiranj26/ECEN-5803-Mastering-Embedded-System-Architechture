Calculates the fluid flow. 

Computes the fluid flow based on various parameters including temperature, density, and viscosity. Uses predefined constants for bluff body width and pipe inner diameter.

Calculates the fluid flow.

The function reads the temperature and based on it calculates the fluid\textquotesingle{}s density and viscosity. Using these values and the Reynold\textquotesingle{}s number, it computes the Strouhal number and uses it to adjust the velocity. Finally, it calculates and sets the flow rate.
