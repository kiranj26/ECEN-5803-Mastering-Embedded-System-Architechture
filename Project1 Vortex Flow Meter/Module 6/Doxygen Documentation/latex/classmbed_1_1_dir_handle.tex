\doxysubsection{Detailed Description}
Represents a directory stream. Objects of this type are returned by a \doxylink{classmbed_1_1_file_system_like}{File\+System\+Like}\textquotesingle{}s opendir method. Implementations must define at least closedir, readdir and rewinddir.

If a \doxylink{classmbed_1_1_file_system_like}{File\+System\+Like} class defines the opendir method, then the directories of an object of that type can be accessed by DIR \texorpdfstring{$\ast$}{*}d = opendir("{}/example/directory"{}) (or opendir("{}/example"{}) to open the root of the filesystem), and then using readdir(d) etc.

The root directory is considered to contain all \doxylink{classmbed_1_1_file_like}{File\+Like} and \doxylink{classmbed_1_1_file_system_like}{File\+System\+Like} objects, so the DIR\texorpdfstring{$\ast$}{*} returned by opendir("{}/"{}) will reflect this.
